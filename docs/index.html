DeskKit

本地小工具集：多文件夹对比与同步、语雀批量导出、Confluence 粘贴用 HTML、重复文件检测等。 基于 Tauri 2 + Rust，安装包更小、启动更快，数据保存在本机。

Tauri 桌面版
下载桌面版 GitHub 源码 使用文档

功能模块

文件夹对比

MD5 / 路径对比，主辅同步、并集、删除与移动。

语雀导出

Token 批量导出知识库，可选 Markdown / Confluence 网页 HTML。仅供个人学习研究，请遵守语雀服务协议，导出内容建议 24 小时内自行删除。

Confluence 转换

Markdown 转 HTML / Word / MD，浏览器复制粘贴保留图片。

重复文件

递归扫描目录，按 MD5 找出重复项。

批量重命名

前缀后缀、替换、序号、清理非法字符。

收藏管理

保存常用文件夹组合，刷新后自动恢复。

开发者本地运行

  1. 安装 Node.js LTS 与 Rust
  2. git clone https://github.com/delbug/deskKit.git && cd deskKit
  3. npm install && npm run tauri:dev
  4. 打包：npm run tauri:build（Mac 双架构：npm run tauri:build:mac）

详细说明见仓库内 使用与打包教程。
