# GitHub Pages 首次启用（只需做一次）

若 Actions 里 **Deploy GitHub Pages** 显示失败，通常是仓库还没打开 Pages。

## 步骤

1. 打开 https://github.com/delbug/deskKit/settings/pages
2. **Build and deployment** → **Source** 选 **GitHub Actions**（不要选 Deploy from a branch）
3. 保存后，打开 **Actions** 标签
4. 选中失败的 **Deploy GitHub Pages** → 右上角 **Re-run all jobs**

## 成功后访问

- 官网地址：**https://delbug.github.io/deskKit/**

## 可选：仓库首页显示官网链接

Settings → General → **Website** 填：`https://delbug.github.io/deskKit/`
